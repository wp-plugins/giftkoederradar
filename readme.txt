== Description ==

Dieses Plugin zeigt dir mit Hilfe einer übersichtlichen Karte alle aktuellen und archivierten Fundorte der Plattform <a href="http://www.giftkoeder-radar.com">Gitfköder Radar</a> direkt in deinem Blog an. Bitte beachte, dass du zum Betrieb dieses Plugins einen <a href="http://www.giftkoeder-radar.com/blog/2011/07/giftkoderradarapi-offenes-okosystem-fur-entwickler/">kostenlosen API Schlüssel</a> benötigst.

Wichtigte Links:

* <a href="http://www.giftkoeder-radar.com">http://www.giftkoeder-radar.com</a>
* <a href="http://www.facebook.com/GiftkoederRadar">http://www.facebook.com/GiftkoederRadar</a>

== Installation ==

* Kopiere das Plugin `gkradar` in das Verzeichnis `/wp-content/plugins/`
* Aktiviere das 'Plugin' im Wordpress Plugin-Menü
* Trage die notwendigen API Schlüssel in den Einstellungen des Plugins ein.
* Verwende den "Shortcode" in deinen Posts (Beispiel: [gkradar])
